## 1. 中文键盘候选栏流程

```mermaid
flowchart TD
    A[Idle状态<br/>显示Logo+工具栏] --> B[用户输入中文编码]
    B --> C[候选显示状态<br/>显示输入候选词]
    
    C --> D{按下空格?}
    D -->|是| E{是否开启联想词?}
    E -->|是| F[联想显示状态<br/>显示联想词候选]
    E -->|否| G[回到Idle状态]
    
    D -->|否| H{按下删除键?}
    H -->|是| I[删除一个编码]
    I --> J{编码是否全部删除完?}
    J -->|是| G
    J -->|否| C
    
    H -->|否| K{继续输入编码?}
    K -->|是| C
    
    F --> L{按下空格?}
    L -->|是| M[返回联想内容候选<br/>继续显示联想词]
    M --> F
    
    L -->|否| N{按下删除键?}
    N -->|是| O[清空联想词候选]
    O --> G
    
    N -->|否| P{用户继续输入中文编码?}
    P -->|是| Q[清除联想词<br/>直接显示新编码的候选词<br/>进入候选显示状态]
    Q --> C
```

**关键规则**
| 场景 | 行为 |
|------|------|
| 中文输入状态下切换键盘 | ✅ 清除编码，回到 Idle |
| 联想状态下继续输入中文 | ✅ 清除联想词，新编码候选直接覆盖显示 |
| 联想状态下按删除 | ✅ 清空联想词，回到 Idle |
| 联想状态下按空格 | ✅ 连续返回联想内容 |

## 2. 英文键盘候选栏流程

```mermaid
flowchart TD
    A[Idle状态] --> B[用户输入英文字符]
    B --> C[英文候选显示状态<br/>显示英文候选词]
    C --> D{按下空格?}
    D -->|是| E[上屏当前候选<br/>回到Idle状态]
    D -->|否| F{按下删除键?}
    F -->|是| G[删除一个字符]
    G --> H{是否还有字符?}
    H -->|是| C
    H -->|否| A
```

## 3. 数字键盘 + 计算器模式流程

```mermaid
flowchart TD
    A[Idle状态<br/>数字键盘] --> B[用户触发计算器模式]
    B --> C[计算器模式<br/>显示计算公式和当前结果]
    C --> D{用户操作}
    D -->|输入数字| E[继续输入数字<br/>追加到公式中<br/>实时更新结果]
    E --> C
    
    D -->|选择结果| F[将最终结果上屏<br/>回到Idle状态]
    
    D -->|退出计算器| A
    
    note right of E
        已上屏的内容不再修改
        只追加新输入的数字
    end note
```

**关键规则：**
- ✅ 计算器模式下可继续输入数字（追加）
- ✅ 已上屏的内容不再修改
- ✅ 选择结果后回到 Idle

## 4. 剪切板显示流程

```mermaid
flowchart TD
    A[用户复制内容] --> B[内容存入剪切板]
    B --> C[打开键盘]
    C --> D{当前是否有输入候选存在?}
    D -->|是| E[不显示剪切板内容<br/>优先显示输入候选]
    D -->|否| F[检测到剪切板有内容<br/>剪切板内容直接显示在候选栏]
    
    F --> G{用户操作}
    G -->|点击复制内容| H[内容上屏<br/>回到Idle状态]
    G -->|用户开始输入| I[隐藏剪切板内容<br/>切换到输入候选状态<br/>进入中文/英文输入流程]
    
    note right of F
        剪切板内容和输入候选
        不会同时存在
    end note
```

**关键规则：**
- ✅ 剪切板内容和输入候选**不会同时存在**
- ✅ 有输入候选时 → 不显示剪切板内容
- ✅ 用户开始输入 → 切换到输入候选状态

## 优先级规则总结

| 优先级 | 显示内容 | 触发条件 |
|-------|---------|---------|
| 1（最高） | 输入候选词 | 用户正在输入中文/英文 |
| 2 | 剪切板内容 | 无输入候选 + 剪切板有内容 |
| 3（最低） | Logo + 工具栏 | 无输入候选 + 无剪切板内容 |
